### Installation
Add this line to your `Cargo.toml`
```toml
rsv = "0.0.5"
```
<br>

**`NO STD` IS NOT INTEGRATED YET**

~~For use in environments where the `std` library is not available you can use this line instead~~

~~This will remove the use of `std` altogether~~
